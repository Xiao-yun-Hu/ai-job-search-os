# How To Use

## Setup

1. Run `npm install` from `/Users/rachelhu/Documents/Claude/Projects/job-search-agent-service`.
2. Run `npm start` from the same directory.
3. Open regular Chrome.
4. Go to `chrome://extensions`.
5. Enable Developer Mode.
6. Click `Load unpacked`.
7. Select `/Users/rachelhu/Documents/Claude/Projects/AI job search plugin/ai-job-search-plugin`.
8. Pin the extension icon to the toolbar.

## Daily Use

1. Start the local service with `npm start`.
2. Open a LinkedIn or BOSS job page in regular Chrome.
3. Open the extension popup.
4. Click `Analyze`, `Apply`, or ask a question in the chat input.
