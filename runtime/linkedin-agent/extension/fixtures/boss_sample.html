入职后的前三个月，你会先梳理现有模型调用、检索、工具执行和日志链路，补齐关键链路的指标体系，并与团队共同确定下一阶段的技术路线。 半年内，我们希望你能主导完成一个稳定的多智能体任务引擎，让 planner、researcher、executor、critic 等不同角色的 agent 可以在受控环境中协作， 支持任务拆解、上下文压缩、长短期记忆、失败重试、人工接管和结果追踪。你需要把架构落到可运行的服务、SDK、后台配置和评测平台中，而不是只停留在概念验证。
