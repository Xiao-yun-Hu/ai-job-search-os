Senior AI Agent Architect

Letta AI
Singapore Remote / Hybrid
SGD 12,000 - 18,000/month

Letta AI is building an AI-native LLM platform for persistent, stateful agents that can reason across tools, memory, and long-running workflows. We are hiring a Senior AI Agent Architect to architect the core agent framework behind our next generation agent platform. The role sits close to product, research, and founding engineering, and will own the end-to-end design of multi-agent systems used by enterprise and developer customers.

You will lead platform-level decisions for agent memory, workflow orchestration, function calling, evaluation pipeline design, RAG integrations, and the boundaries between the agent runtime and external tools. The work includes designing APIs for developers, building Python and TypeScript services, improving observability for autonomous decision systems, and creating evaluation infra that lets teams compare agent behavior across model versions, prompts, datasets, and production traces.

We are a Series B company with a focused 10-80 person team, backed by investors who understand deep technical infrastructure. This is not a generic chatbot role. You will collaborate with ML engineers, product managers, and customer-facing engineers to turn ambiguous requirements into durable agent OS capabilities, including memory substrate design, tool permissioning, task planning, retrieval quality, and multi-agent coordination for complex business workflows.

The ideal candidate has 8+ years of software engineering experience, including several years as a technical lead, staff engineer, principal engineer, or platform architect. You should have shipped distributed systems, worked directly with LLM integration, vector database and embedding pipelines, and made pragmatic choices about reliability, latency, cost, and security. Experience with LangChain, LangGraph, AutoGen, Kubernetes, FastAPI, cloud infrastructure, model evaluation, or fine-tuning is useful, but we care more about judgment and ownership than a specific stack.

This position is based in Singapore with remote and hybrid flexibility. We work async where possible, keep meetings intentional, and expect senior engineers to communicate clearly across design docs, code reviews, and customer architecture discussions. You will have high agency, meaningful equity exposure, and responsibility for systems that define how developers build reliable agentic applications on top of modern foundation models.
