HERMES SETUP  ·  2026.05

在 Hermes 上
怎么用

1
装 skill
git clone ai-job-search-skill

bash install.sh

自动创建数据目录 + 注册 chrome-devtools MCP
2
启动 Chrome（CDP 模式）
open -a "Google Chrome" --args --remote-debugging-port=9222

用你已登录的日常 Chrome，不要新 profile
3
首次运行 → 自动 Onboarding
在 Hermes 输入：
/skills run ai-job-search "morning outreach"

档案为空时自动进入引导，约 50 分钟完成设置
4
每天早上一句话
skill 自动加载你的档案 → Scout 岗位 → Match 评级 → 发打招呼
Tier A 高优先级   Tier B 正常投递   Tier C/D 跳过
💡 Memory bootstrap：每次对话开始，skill 先读取你的完整档案和近 3 天复盘， 新对话不丢上下文，AI 知道你上次投了什么、哪里有回音
